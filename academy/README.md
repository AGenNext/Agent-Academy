# AGenNext Academy

AGenNext Academy is the learning and certification layer for enterprise agent engineering.

It organizes courses, labs, projects, and certifications into role-based learning paths. The Academy is repo-native first: every course is versioned, reviewable, forkable, and connected to real engineering artifacts.

## Academy Mission

Help builders move from AI prompting to governed agent engineering.

AGenNext Academy teaches how to build agent systems that are:

- production-ready
- source-grounded
- policy-aware
- auditable
- testable
- secure by design
- cost-aware
- human-reviewable
- enterprise-operable

## Learning Model

```text
Concept -> Lab -> Project -> Evidence -> Review -> Certification
```

Every learner should leave with working artifacts, not only notes.

## Academy Promise

A course is complete only when it provides:

1. Clear outcomes
2. Structured lessons
3. Hands-on labs
4. Reviewable projects
5. Evidence requirements
6. Governance notes
7. Certification or completion criteria
8. Maintenance owner

## Operating Principle

AI can accelerate learning. Evidence makes learning real.
