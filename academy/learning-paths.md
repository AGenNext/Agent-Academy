# Learning Paths

Learning paths organize courses into outcomes for specific roles.

## Path 100 — Agent Practitioner

For builders who want to use agent tools responsibly.

### Outcomes

- Understand agent concepts
- Use AI tools with clear boundaries
- Write good task briefs
- Verify output with evidence
- Recognize unsafe automation

### Recommended Courses

1. SurrealDB Foundations
2. Agent Memory
3. Claude as a Course: Modules 01-03
4. Agent Skills & Blueprints

### Completion Evidence

- One task brief
- One `CLAUDE.md`
- One simple lab submission
- One reflection on safe tool use

## Path Rule

Every path must end with evidence. Learning without evidence is not certification-ready.
